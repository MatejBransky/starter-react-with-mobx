# starter-react-with-mobx (TypeScript)

## Options

- indentation, tab vs spaces and single/double quotes: in `.prettierrc`:

```json
{
  "singleQuote": true,
  "useTabs": false,
  "tabWidth": 2
}
```

- browser compatibility: in `.browserslistrc` ([more info](https://github.com/browserslist/browserslist))
